# Study-Spring-Security
`Spring Boot`에서 제공하는 `Spring Security`를 공부하기 위한 저장소

## 학습 목표
- `Spring Security`의 이해
- `Spring Session` 기반의 로그인 구현
- `JWT` 기반의 로그인 구현
- `Spring OAuth2` 기반의 소셜 로그인 구현
